---------------------------------
Famicom Setup Instructions [fcsi]
---------------------------------

1. Required Files
	1a. Nintemulator relies on a ROM database to obtain the most accurate information pertaining to a ROM. This information is needed for many games,
		and the standard iNES header is insufficient for this purpose. The specific database used by Nintemulator is available for download at
		http://bootgod.dyndns.org:7777/xml.php.

2. Setup Steps
	2a. Place the XML database into {Nintemulator Directory}/FC/database.xml, renaming it as necessary.

Game Boy Setup Instructions [gbsi]
----------------------------------

1. Required Files
	1a. Nintemulator requires the use of the DMG Bootstrap ROM for an authentic experience. This file can be dumped from a real Game Boy.

2. Setup Steps
	2a. Place the bootstrap ROM into {Nintemulator Directory}/GB/boot.rom, renaming it as necessary.
